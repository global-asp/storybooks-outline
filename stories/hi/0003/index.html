स्कूल के कपड़े School clothes Les vêtements d’école

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Tanvi Sirari

Hindi

Level 1

ये मिडी लंबी है।

This dress is long.

Cette robe est longue.

ये स्वेटर बड़ी है।

This jersey is big.

Ce pull est grand.

ये बस्ता बड़ा है।

This bag is big.

Ce sac est grand.

ये पेटी…

This belt is...

Cette ceinture est...

ये टोपी छोटी है।

This hat is small.

Ce chapeau est petit.

ये मोजे छोटे है।

The socks are short.

Les chaussettes sont courtes.

लेकिन ये जूते नये है।

But these shoes are new.

Mais ces souliers sont neufs.

…ये सही नाप के है।

... they fit well.

... ils font bien.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Tanvi Sirari
Language: Hindi
