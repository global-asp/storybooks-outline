我嘅身體 My body Mon corps

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

dohliam

Zoe Lam

Cantonese

Level 1

我可以同佢跑。

I can run with it.

Je peux courir avec.

我可以同佢跳。

I can jump with it.

Je peux sauter avec.

我可以同佢跳舞。

I can dance with it.

Je peux danser avec.

我可以同佢游水。

I can swim with it.

Je peux nager avec.

我可以同佢跳繩。

I can skip with it.

Je peux jouer à la corde à danser avec.

我可以同佢踢波。

I can kick with it.

Je peux donner un coup de pied avec.

我可以同佢逃走。

I can escape with it.

Je peux échapper avec.

但係永遠都唔可以同佢飛！

But I can never fly with it.

Mais je ne peux jamais voler avec.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: dohliam
Read by: Zoe Lam
Language: Cantonese
Level 1
Source: My body from African Storybook
